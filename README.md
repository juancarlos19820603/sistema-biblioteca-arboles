# Sistema de Gestión de Biblioteca con Árboles AVL

Sistema completo de gestión de biblioteca con interfaz gráfica y estructuras de datos optimizadas.

## Características
- Gestión de libros, usuarios y préstamos
- Búsquedas optimizadas con árboles AVL
- Interfaz gráfica con Tkinter
- Reportes estadísticos avanzados
- Persistencia de datos en JSON

## Estructuras de datos implementadas
- Árboles AVL para búsquedas rápidas
- Listas enlazadas para almacenamiento
- Árbol de segmentos para reportes

```bash
git clone https://github.com/tu-usuario/sistema-biblioteca.git
